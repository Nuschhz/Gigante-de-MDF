◆ dano()

void dano ( )

Função que administra o dano recebido pelo Gigante de MDF.

Esta Função contabiliza os danos recebidos pelo Gigante. Para cada dano recebido, é alternado o BIT dos leds de vida para que estas sejam decrementadas e o laser do Gigante é desativado. Ao receber 3 danos, são executadas as funções de giro e desliga.
